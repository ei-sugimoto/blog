## 機能紹介

### コードハイライト

JavaScript のサンプルコード：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to our tech blog, ${name}!`;
}

const message = greet('World');
```

TypeScript の例：

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

const createUser = (userData: User): User => {
  return {
    ...userData,
    id: Date.now(),
  };
};
```
